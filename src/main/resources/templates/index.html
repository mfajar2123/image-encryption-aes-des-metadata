Encryption and Decryption Algorithms

This system employs the AES-128 algorithm to encrypt, ensuring robust security for image data. For an extra layer of protection, DES permutation is used as the key to encrypt images with AES-128. The encrypted result is discreetly embedded within the image metadata, serving as a component of steganography.
